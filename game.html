Game Project

This is a little game about collecting items and placing all the items in a single place to win.

Game

In this console game you will be in a town then will have to go to different towns to collect items and place all items in a single town to win. You will have a limited amount of lives and moves. Everytime you move to a different town you will lose a move and once you have zero moves you will lose. The image above will show you the map of the towns you will start on KevTown. The best way to win will be once you start you will have to go to NedTown then back to KevTown drop the item you picked from NedTown. Then go to HashTown get the item go to FredTown get the items go back to kevTown drop the items. Then make your way to EdTown get the last items then finaly go back to KevTown and drop the last items to win. To see the project you can visit the Game Repository to download the project.
