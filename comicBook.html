Comic Book Store

This is a small comic book store app where a user will be able to create an account and login. Any user without an account will be able to see the items in the guest option. Once an account is created the program will save the information in a local data base created by the program. In the program the users will be able to add items to a cart and edit the cart. Finaly users will be able to go to the checkout page to finalise the purchase. There is also a manager view where users who are employees will be able to manage customers, employees, and items. Being able to edit item, employees or customer information and accounts. To see more information or download the project you can see this Repository
